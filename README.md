# DeltaWI
Dyalog APL emulator for ⎕WI

**Currently under initial development**

This project is a replacement for the ⎕WI system function to be used in Dyalog APL. It should function as closely as possible to the original system function.

Documentation for its use and internal construction will eventually be stored in MarkDown files here, but for now is available in PDF form upon request.

Any comments, fixes, or new code can be managed with the usual GitHub mechanisms (e.g. Pull Requests).
